## 安装说明 / Installation

### 🪟 Windows

1. 点上面的「下载 Windows 版」，下载 `OpenWiki_0.3.14_x64-setup.exe`。
2. 双击运行。若弹出「Windows 已保护你的电脑」蓝色提示，点 **更多信息 → 仍要运行**（本程序暂未做 Windows 代码签名，这是正常提示，文件是安全的）。
3. 按提示完成安装。打开后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。

### 🍎 macOS

1. 按你的芯片，点上面对应的下载链接。
2. 双击 `.dmg`，将 **OpenWiki** 拖入 **Applications**（应用程序）文件夹，覆盖旧版本。
3. 双击 OpenWiki 即可打开（已做 Apple 签名 + 公证，不会被拦截）。
4. 打开后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。
