# OpenWiki

> 📋 捕获碎片信息，构建个人知识库

OpenWiki 是一个 Mac 桌面应用，帮你自动收集浏览过程中看到的有价值内容，用 AI 构建个人知识库并生成洞察报告。

## ✨ 功能特性

### 🔍 自动捕获
- 监听剪贴板，自动保存复制的内容
- 支持文本、图片、URL 链接
- 自动识别来源应用

### 📂 内容管理
- 按类型过滤（文本/图片/链接）
- 本地 SQLite 存储，隐私安全
- 快速检索历史内容

### 📊 AI 周报生成
- 一键生成本周内容总结
- 自动归类、提炼要点
- 支持反馈优化（点赞/踩）

### ⌨️ 快捷操作
- 系统托盘常驻
- 全局快捷键唤起
- 后台静默运行

## 🎯 使用场景

- **信息收集党**：每天刷推、看文章，想保存精华但懒得整理
- **内容创作者**：收集灵感素材，定期回顾生成创作思路
- **知识管理者**：碎片化学习，定期汇总沉淀

## 🛠 技术栈

| 层级 | 技术 |
|---|---|
| 框架 | Tauri 2 |
| 前端 | React + TypeScript |
| 样式 | Tailwind CSS |
| 动效 | Framer Motion |
| 存储 | SQLite (本地) |
| 后端 | Rust |

## 📦 安装

```bash
# 克隆仓库
git clone https://github.com/kdsz001/xiaoyun.git
cd xiaoyun

# 安装依赖
pnpm install

# 开发模式
pnpm tauri dev

# 构建应用
pnpm tauri build
```
